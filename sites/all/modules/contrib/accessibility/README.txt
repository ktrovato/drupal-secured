
-- SUMMARY --

This is a suite of modules that check content, themes, and rendred code for accessibility problems.

For a full description visit the project page:
  http://drupal.org/project/accessibility
Bug reports, feature suggestions and latest developments:
  http://drupal.org/project/accessibility/libraries
False positives are usually due to the QUAIL jQuery library, file an issue here:
  http://github.com/kevee/quail


-- REQUIREMENTS --

* Libraries module - http://drupal.org/project/libraries
* QUAIL jQuery Library - http://github.com/kevee/quail/releases (to be installed in your libraries directory, usually sites/all/libraries, make sure to rename the folder to just "quail" if you download a release with a version number appended)


-- INSTALLATION --

* Install like any other module, see http://drupal.org/node/70151 for more help.
