===============================
All links must have a "title" attribute
===============================

*Severity code:* Severe error

.. php:class:: aMustHaveTitle


Every link must have a "title" attribute.



Example
-------
Wrong
-----

.. code-block:: html

    <a href="home.html">Home</a>



Right
-----

.. code-block:: html

    <a href="home.html" title="Return to home">Home</a>
