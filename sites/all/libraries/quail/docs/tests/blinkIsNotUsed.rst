===============================
The "blink" tag should not be used
===============================

*Severity code:* Severe error

.. php:class:: blinkIsNotUsed


The <code>blink tag should not be used. Ever.



Example
-------
Wrong
-----

.. code-block:: html

    <blink>Please read me!</blink>



Right
-----

.. code-block:: html

    <strong>Please read me!</strong>
