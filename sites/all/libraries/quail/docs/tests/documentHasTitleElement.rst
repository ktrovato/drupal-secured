===============================
The document should have a title element
===============================

*Severity code:* Severe error

.. php:class:: documentHasTitleElement


The document should have a title element.



Example
-------
Wrong
-----

.. code-block:: html

    <html lang="en"><head></head><body>....



Right
-----

.. code-block:: html

    <html lang="en"><head><title>My Great Web page</title></head><body>....
