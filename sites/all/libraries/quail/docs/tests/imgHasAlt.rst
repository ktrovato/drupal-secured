===============================
Image elements must have an "alt" attribute
===============================

*Severity code:* Severe error

.. php:class:: imgHasAlt


All <code>img elements must have an alt attribute



Example
-------
Wrong
-----

.. code-block:: html

    <img src="dog.jpg">



Right
-----

.. code-block:: html

    <img src="dog.jpg" alt="A photograph of a dog">
