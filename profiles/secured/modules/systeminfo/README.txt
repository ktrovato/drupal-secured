
Module
======
System Information


Description
===========
This module displays information about the current state of the Drupal
installation and system environment.


Requirements
============
Drupal 7.x


Installation
============
1. Move this folder into your modules directory.
2. Enable it from Administration >> Modules >> Monitoring.


Configuration & Using
=====================
View the information at Administration >> Reports >> System information.

For more information, see the online handbook entry at
http://drupal.org/node/1225540
